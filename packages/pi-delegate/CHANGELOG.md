# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-06-25

### Added

- Initial release of pi-delegate
- Parent agent interface for spawning and orchestrating child Pi agents
- Delegate provider for agent lifecycle management
- Support for parallel and sequential agent execution modes
- Agent input/output validation using TypeBox schemas
- YAML configuration support for agent definitions
- TypeScript type definitions for all public APIs
- Comprehensive test suite with 71 test cases
- Full compatibility with Pi 0.79.8 and pi-coding-agent 0.79.9
